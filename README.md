# electron-app

## Automatically install and run

```bash
./start.sh
```

## Automatically

```bash
./install.sh
./run.sh
```

## Manually

### Pre-installed depences

```bash
apt update
apt install -y nodejs npm python3 python3-pip
```

### Server

```bash
cd server
pip3 install -r requirements.txt
python3 server.py
```

### Client

```bash
cd client
npm install
npm run start
```
